## 📝 功能说明

### 1. 登录页面
- 访问 `/login.html` 进入登录页面
- 输入用户名和密码登录
- 登录成功后自动跳转到管理后台

### 2. 管理后台
- 登录后可访问 `/admin.html` 管理后台
- 未登录访问会自动重定向到登录页面
- 所有数据修改操作都需要登录权限

### 3. 会话管理
- 登录会话有效期: 24小时
- 会话过期后需要重新登录
- 点击右上角"退出登录"按钮可主动退出

## 🔧 修改密码

如需修改管理员密码，请按以下步骤操作:

1. 在服务器上生成新密码的哈希值:
```bash
cd /var/www/x-lab/server
node generate-password.js 你的新密码
```

2. 复制生成的哈希值

3. 编辑 `server/server.js` 文件:
```javascript
const ADMIN_CREDENTIALS = {
  username: 'admin',
  passwordHash: '这里粘贴新的哈希值'
};
```

4. 重启应用:
```bash
pm2 restart x-lab-website
```

## ❓ 常见问题

**Q: 登录后显示"会话已过期"怎么办？**  
A: 这是正常的会话超时，重新登录即可。

**Q: 忘记密码怎么办？**  
A: 需要在服务器上重新生成密码哈希并修改配置文件。

**Q: 可以添加多个管理员吗？**  
A: 目前系统只支持单个管理员账号。如需多账号支持，需要修改代码实现用户管理功能。
